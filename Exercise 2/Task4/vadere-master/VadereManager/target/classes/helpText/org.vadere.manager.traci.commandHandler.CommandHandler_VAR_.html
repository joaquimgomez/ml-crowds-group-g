CommandHandler

CommandHandler classes perform the actual request within the command. See CommandExecutor on how commands are dispatched to the correct CommandHandler subclass. These classes implement methods which adhere to the TraCICmdHandler Interface. These methods are used by the CommandExecutor for dispatching.
