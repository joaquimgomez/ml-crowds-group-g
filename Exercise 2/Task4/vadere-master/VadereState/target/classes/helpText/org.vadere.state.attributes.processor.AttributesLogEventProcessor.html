AttributesLogEventProcessor

---

Field: compartmentIndex

The compartment to observe. Index 0 and the last index denote half-compartments which should not be observed.

Field: firstLogEventId

Log event ID for initializing the ID counter to prevent duplicate log event IDs from different log event processors.

Field: personIdOffset

Offset to add to the pedestrian ID in order to prevent duplicate person IDs from different log event processors.

Field: surveyId
