Trajectory

A Trajectory is a list of org.vadere.state.scenario.Pedestrian objects, that can be seen as pedestrian states of the same pedestrian. The representing pedestrian is the same, so all org.vadere.state.scenario.Pedestrian objects has the same id but the state of a pedestrian changes over time. @author Benedikt Zoennchen
