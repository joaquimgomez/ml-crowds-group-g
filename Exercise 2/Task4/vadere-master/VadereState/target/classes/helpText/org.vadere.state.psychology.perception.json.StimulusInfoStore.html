StimulusInfoStore

This class bundles multiple @see StimulusInfo objects. This class is just a wrapper to get a convenient JSON de-/serialization. The JSON serialization should look like this: "stimulusInfos": [ { "timeframe": { "startTime":..., "endTime":..., "repeat":..., "waitTimeBetweenRepetition":... }, "stimuli": [ {"type":"ElapsedTime" }, {"type":"WaitInArea", "area": ... }, ... ] }, { ... } ]
