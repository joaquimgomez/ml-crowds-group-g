TestAttributesSerializability

Checks that every Attribute is serializable. All classes in the attribute package (and below that package) will be included.

Field: pathToAttributePackage

The packagePath to the folder that contain all Attribute-Classes or that contain folder that contain Attribute-Classes.

Field: packagePath

The package packagePath to the folder (starting at /classes) ontain all Attribute-Classes or that contain folder that contain Attribute-Classes.

Field: checkClass

Checks if minimum one class is checked.
