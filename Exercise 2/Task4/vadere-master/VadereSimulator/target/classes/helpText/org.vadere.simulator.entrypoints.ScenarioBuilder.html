ScenarioBuilder

ScenarioBuilder provide methods to manipulate the attributes/parameters of a scenario. It's an interface for generate multiple Scenarios based on the same infrastructure but with different attributes/parameters. The class does not provide methods to change the infrastructure of the org.vadere.state.scenario.Topography. To use this effectively the user need a good knowledge about the parameters of the given scenario! @author Benedikt Zoennchen
