PotentialFieldTarget

@author Benedikt Zoennchen

Field: logger

---

Field: potentialFieldsNeedUpdate

false if and only if there exits no dynamic potential field.

Field: domain

the topography of the running simulation.

Field: attributes

configuration of the potential fields.

Field: eikonalSolvers

Stores all potential fields which represent to a target (targetId). This Map has to be filled by classes extending this class.

Field: attributesPedestrian

configuration of the agents.

Field: lastUpdateTimestamp

the simulation time in seconds in which the last update was performed.
