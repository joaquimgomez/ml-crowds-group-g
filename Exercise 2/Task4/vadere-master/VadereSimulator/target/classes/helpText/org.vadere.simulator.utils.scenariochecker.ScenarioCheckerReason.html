ScenarioCheckerReason

This enum collects all reasons for ScenarioCheckerMessages. The label is the messageId for locale de/en. This enum is needed, because the messages are part or the GUI module and a dependency on this module would create a cyclic dependency. A possible solution would be a separated module only for locale messages.
